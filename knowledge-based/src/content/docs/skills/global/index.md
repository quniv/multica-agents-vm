---
title: Global skills
description: Reusable agent guidance that applies across Multica projects and repositories.
---

Global skills document stable, broadly applicable workflows. They are rendered
Starlight guidance, not executable `SKILL.md` artifacts. The source of truth
for each published skill is this version-controlled documentation and its
reviewed pull request.

Publish a global skill only after the guidance has an identified reviewer or
owner, is safe to share across repositories, and is useful beyond the task
that prompted it.
