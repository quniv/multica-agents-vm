# Git

Document repository-specific Git workflows and conventions.

## Entry format

Store each Git topic at `git-NNN_<topic-name>/index.md`, such as
`git-001_release-branches/index.md`.

- Code: The next unused three-digit code, starting with `git-001`.
- Description: A concise summary of the workflow or convention.

Add the topic to this index when creating it. Keep its code and description
synchronized with the Git page.
