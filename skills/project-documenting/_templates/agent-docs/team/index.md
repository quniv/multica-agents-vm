# Team

Track team members, roles, and ownership.

## Entry format

Store each member at `member-NNN_<member-name>/index.md`, such as
`member-001_jane-doe/index.md`.

- Code: The next unused three-digit code, starting with `member-001`.
- Name: The member's preferred name.
- Description: A concise summary of their responsibilities.
- Role: Their project role.

Add the member to this index when creating the page. Keep its code, name,
description, and role synchronized with the member page.
